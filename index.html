What are Regular Expressions?

Regular expressions (regex) are patterns used to match character combinations in strings. 


				  // Example: matching a simple pattern
				  const regex = /abc/;
				  const result = regex.test('abcdef'); // returns true

Using Basic Symbols in Regular Expressions

Here are some of the most common symbols in regular expressions:

  • . matches any single character except newline.
  • * matches 0 or more repetitions of the preceding character.
  • + matches 1 or more repetitions of the preceding character.
  • ^ matches the start of the string.
  • $ matches the end of the string.

Examples:


				  // Example 1: Using "."
				  const regexDot = /a.c/;
				  console.log(regexDot.test('abc')); // returns true
				  console.log(regexDot.test('ac')); // returns false

				  // Example 2: Using "*"
				  const regexStar = /ab*c/;
				  console.log(regexStar.test('ac')); // returns true
				  console.log(regexStar.test('abbc')); // returns true

				  // Example 3: Using "+"
				  const regexPlus = /ab+c/;
				  console.log(regexPlus.test('abc')); // returns true
				  console.log(regexPlus.test('ac')); // returns false

				  // Example 4: Using "^" and "$"
				  const regexStartEnd = /^abc$/;
				  console.log(regexStartEnd.test('abc')); // returns true
				  console.log(regexStartEnd.test('abc123')); // returns false

Meta Characters in Regular Expressions

These are special characters that help you create more complex regex patterns:

  • \d matches any digit (0-9).
  • \w matches any word character (alphanumeric + underscore).
  • \s matches any whitespace character (spaces, tabs, newlines).
  • \b matches a word boundary.

Example 1: Using "\d"


	  // Matches any digit (0-9)
	  const regexDigit = /\d/;
	  console.log(regexDigit.test('123')); // returns true (contains digits)
	  console.log(regexDigit.test('abc')); // returns false (no digits)

Example 2: Using "\w"


	  // Matches any alphanumeric character (letters, numbers, underscores)
	  const regexWordChar = /\w/;
	  console.log(regexWordChar.test('abc123')); // returns true (valid word characters)
	  console.log(regexWordChar.test('!@#')); // returns false (special characters)

Example 3: Using "\s"


	  // Matches any whitespace character (space, tab, newline)
	  const regexSpace = /\s/;
	  console.log(regexSpace.test('hello world')); // returns true (contains space)
	  console.log(regexSpace.test('helloworld')); // returns false (no space)

Example 4: Using "\b"


	  // Matches a word boundary
	  const regexWordBoundary = /\bcat\b/;
	  console.log(regexWordBoundary.test('the cat is here')); // returns true (word boundary)
	  console.log(regexWordBoundary.test('the catapult is here')); // returns false (no boundary)

Quantifiers in Regular Expressions

Quantifiers specify how many times an element must occur:

  • * — 0 or more times
  • + — 1 or more times
  • ? — 0 or 1 time
  • {n} — exactly n times
  • {n,} — n or more times
  • {n,m} — between n and m times

Example 1: *


        // Matches 'a' repeated 0 or more times
        const regex = /a*/;
        console.log('aaa'.match(regex)); // ['aaa']
        console.log('bbb'.match(regex)); // ['']

Example 2: +


        // Matches 'a' repeated 1 or more times
        const regex = /a+/;
        console.log('aaa'.match(regex)); // ['aaa']
        console.log('bbb'.match(regex)); // null

Example 3: ?


        // Matches 'a' 0 or 1 time
        const regex = /a?/;
        console.log('a'.match(regex)); // ['a']
        console.log('b'.match(regex)); // ['']

Example 4: {n}


        // Matches exactly 3 'a's
        const regex = /a{3}/;
        console.log('aaab'.match(regex)); // ['aaa']
        console.log('aab'.match(regex)); // null

Example 5: {n,}


        // Matches 2 or more 'a's
        const regex = /a{2,}/;
        console.log('aaaaa'.match(regex)); // ['aaaaa']
        console.log('a'.match(regex)); // null

Example 6: {n,m}


        // Matches between 2 and 4 'a's
        const regex = /a{2,4}/;
        console.log('aaaaa'.match(regex)); // ['aaaa']
        console.log('a'.match(regex)); // null

Character Classes in Regular Expressions

Character classes allow you to match specific sets or ranges of characters:

  • [abc] — matches any one of 'a', 'b', or 'c'
  • [a-z] — matches any lowercase letter
  • [^abc] — matches any character except 'a', 'b', or 'c'

Example 1: [abc]


        // Matches 'a', 'b', or 'c'
        const regex = /[abc]/;
        console.log('apple'.match(regex)); // ['a']
        console.log('dog'.match(regex)); // null

Example 2: [a-z]


        // Matches any lowercase letter
        const regex = /[a-z]/;
        console.log('Hello'.match(regex)); // ['e']
        console.log('123'.match(regex)); // null

Example 3: [^abc]


        // Matches any character except 'a', 'b', or 'c'
        const regex = /[^abc]/;
        console.log('cat'.match(regex)); // ['t']
        console.log('aaa'.match(regex)); // null
